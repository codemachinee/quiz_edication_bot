# Инструкции для Gemini

Этот файл содержит инструкции и соглашения для работы с AI-ассистентом Gemini в данном проекте.

## Общение

- Все ответы и вопросы должны быть на русском языке.

## Обзор проекта

Это образовательный Telegram-бот для проведения викторин, написанный на Python. Основная логика построена на принципе конечных автоматов (FSM).

## Управление зависимостями

Проект использует `uv` для управления зависимостями.

- Для установки или обновления всех зависимостей из `pyproject.toml` выполните:
  ```shell
  uv sync
  ```

## Запуск приложения

- Для запуска основного файла бота используйте:
  ```shell
  python main.py
  ```

## Структура и архитектура

- `main.py`: Точка входа в приложение.
- `handlers.py`: Содержит обработчики сообщений и колбэков от Telegram.
- `FSM.py`: Определяет состояния для конечного автомата (Finite State Machine).
- `keyboards.py`: Содержит функции для создания различных клавиатур бота.
- `functions.py`: Вспомогательные функции, используемые в разных частях проекта.
- `structure.py`: Определяет структуру данных, например, вопросы для викторин.

## Стиль кода и линтинг

Проект использует `ruff` для линтинга и форматирования кода.

- Перед коммитом убедитесь, что код отформатирован:
  ```shell
  ruff format .
  ```
- И не содержит ошибок линтера:
  ```shell
  ruff check .
  ```
